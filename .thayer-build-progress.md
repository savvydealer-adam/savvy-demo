## How to Resume

1. Read this file first
2. For each MISSING page: read the existing index.html of that template to get the header/footer/CSS, then create the missing page matching the same patterns
3. For financing pages: hero, advantage cards, pre-qual form, FAQ accordion
4. For contact pages: Google Maps iframe (Bowling Green OH), info cards with address/phone, hours tables, contact form
5. After all pages built: update gallery.html, commit, push, deploy
6. Deploy command: `gcloud builds submit --project=savvydealer-website --config=cloudbuild.yaml --substitutions=COMMIT_SHA=$(git rev-parse --short HEAD)`
