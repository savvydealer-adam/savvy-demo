# Ford Vehicle Image URLs for Dealership Demo Sites

All URLs verified accessible (HTTP 200) on 2026-03-14. No authentication required.

## URL Pattern Notes

**Two CDN sources:**
1. `https://www.ford.com/acslibs/content/dam/...` -- Jellybean/product shots (PNG, transparent backgrounds)
2. `https://www.assets.ford.com/adobe/assets/...` -- Hero/lifestyle photography (WebP, scene backgrounds)

**assets.ford.com supports query params:** `?width=1920&max-quality=90` for size/quality control.

**Image angle codes:**
- `dr34` = driver-side 3/4 front (best for hero shots)
- `dr78` = driver-side 3/4 rear
- `ps34` = passenger-side 3/4 front (smaller/thumbnail)
- `ds34` = driver-side 3/4 (Maverick variant)
